### Obejtivo del Repositorio
Este espacio ha sido diseñado para centralizar y exhibir el progreso técnico de los estudiantes en el submódulo de **Implementa software de sistemas informáticos**. Más que una carpeta de entrega, este repositorio funciona como un **Portafolio Profesional** que documenta la evolución del alumno desde sus primeras líneas de código hasta aplicaciones complejas.

### 🛠️ Competencias a Desarrollar
Al finalizar el ciclo, el estudiante habrá demostrado dominio en:
* **Control de Versiones:** Uso profesional de Git (Commits, Branches, Pull Requests).
* **Lógica de Programación:** Resolución de problemas mediante algoritmos.
* **Documentación Técnica:** Redacción de archivos Markdown y comentarios de código.
* **Gestión de Proyectos:** Organización jerárquica de archivos y recursos.\
